# e-plantShopping

e-plantShopping is a React-based web application for browsing and shopping for plants online.  
This project demonstrates React components, state management, routing, and modern frontend development practices.

## Features

- Browse different plants
- Add plants to cart
- Responsive user interface
- React component architecture

## Technologies Used

- React
- JavaScript
- CSS
- Node.js
